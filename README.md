# ETH Flow

Ethereum [reactflow](https://reactflow.dev/) components.

## Getting started

```bash
# install dependencies
$ yarn

# 1. Terminal
# start local chain
$ yarn chain

# 2. Terminal
# deploy contracts
$ yarn deploy

# start app
$ yarn start
```
